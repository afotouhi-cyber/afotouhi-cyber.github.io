## Python System Security Scan Report

![Python security scan report overview](/assets/images/VulnReport.png)

![Python security scan vulnerable ports section](/assets/images/Vuln2.png)

Built a Python-based security reporting script that collects system information and highlights potential security concerns such as outdated applications, exposed ports, running processes, and system configuration details.

### What It Checks

- System platform, architecture, hostname, CPU, memory, and disk usage
- Installed application versions and available updates
- Exposed or potentially vulnerable ports such as SMB/NetBIOS
- Running process samples
- Basic CVE summary output when available

### Skills Demonstrated

`Python` · `Security Automation` · `Windows Security` · `System Enumeration` · `Port Review` · `Vulnerability Awareness` · `HTML Reporting`
